# Witryna do organizacji wydarzeń

- Aplikacja pozwala na zorganizowanie wydarzeń, które posiadają pewną ograniczoną ilość miejsc a przez to wymagają posiadania biletu (płatnego lub darmowego).
- Dowolny użytkownik może utworzyć wydarzenie, dodać szczegółowy opis wraz z grafikami itp.. Miejsce wydarzenia można podglądnąć na mapie oraz w kalendarzu. Wydarzenie można dodać do swojego kalendarza.
- Każde wydarzenia posiada jedno lub więcej wystąpień. Na każde wystąpienie można zakupić bilet. Po kupnie biletu możliwe jest jego wyeksportowanie w formacie PDF.
